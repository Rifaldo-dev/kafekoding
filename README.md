# KafeKoding Website

Website resmi komunitas KafeKoding - "Kami Memilih Turun Tangan"

## 🚀 Tentang

KafeKoding adalah komunitas belajar teknologi informasi di Sumatera Barat yang berfokus pada pengembangan kemampuan anggota secara individu maupun tim sesuai kebutuhan industri.

## ✨ Fitur

- **Halaman Utama** - Hero section, statistik, tentang kami, kelas, mentor, blog
- **Form Pendaftaran** - Pendaftaran anggota dengan validasi lengkap
- **Blog/Artikel** - Aktivitas dan artikel komunitas
- **Responsive Design** - Optimized untuk desktop dan mobile
- **Modern Tech Stack** - React, Vite, Tailwind CSS

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite 8
- **Styling**: Tailwind CSS 3
- **Icons**: Lucide React, FontAwesome
- **Routing**: React Router DOM 7
- **Build Tool**: Vite
- **Deployment**: Vercel

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/Rifaldo-dev/kafekoding.git
cd kafekoding

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Features

### Halaman Utama
- Hero section dengan animasi typing
- Statistik komunitas
- Informasi tentang komunitas
- Daftar kelas yang tersedia
- Profil mentor dan tim
- Blog/artikel terbaru

### Form Pendaftaran
- Validasi form lengkap
- Dropdown institusi (Metamedia, YPTK, UNP, dll)
- Pilihan kelas dengan validasi waktu
- Upload bukti follow Instagram
- Success message dengan detail pendaftaran

### Blog
- Daftar artikel dan aktivitas
- Search functionality
- Pagination
- Responsive card layout

## 🌐 Deployment

Website ini di-deploy menggunakan Vercel dengan konfigurasi:

- SPA routing configuration
- Security headers
- Asset optimization
